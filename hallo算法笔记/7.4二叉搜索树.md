# 7.4 二叉搜索树（BST）

BST 是有规律的二叉树：**左子树所有值 < 根 < 右子树所有值**，递归成立。

```
        8
       / \
      3   10
     / \    \
    1   6   14
       / \
      4   7
```

## 查找

```java
TreeNode search(TreeNode root, int target) {
    if (root == null || root.val == target) return root;
    if (target < root.val) return search(root.left, target);   // 去左边
    return search(root.right, target);                          // 去右边
}
// 每次排除一半，时间复杂度 O(log n)（平衡时）
```

## 插入

```java
TreeNode insert(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);  // 找到空位，插入
    if (val < root.val)  root.left  = insert(root.left, val);
    else if (val > root.val) root.right = insert(root.right, val);
    // val == root.val 时不插（不允许重复）
    return root;
}
```

## 删除（三种情况）

```
情况1：被删节点无子节点 → 直接删
情况2：被删节点有一个子节点 → 子节点顶上来
情况3：被删节点有两个子节点 → 用中序后继（右子树最小值）替换
```

```java
TreeNode remove(TreeNode root, int val) {
    if (root == null) return null;
    if (val < root.val) {
        root.left = remove(root.left, val);
    } else if (val > root.val) {
        root.right = remove(root.right, val);
    } else {
        // 找到了，执行删除
        if (root.left == null)  return root.right;  // 情况1、2
        if (root.right == null) return root.left;   // 情况2
        // 情况3：找右子树最小值（最左节点）
        TreeNode min = root.right;
        while (min.left != null) min = min.left;
        root.val = min.val;                          // 替换值
        root.right = remove(root.right, min.val);   // 删除那个最小节点
    }
    return root;
}
```

## BST 的性能

| 操作 | 平均（平衡） | 最坏（退化成链表） |
|------|------------|----------------|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

> 按顺序插入 1,2,3,4,5 会退化成链表！这就是需要 AVL 树的原因。
