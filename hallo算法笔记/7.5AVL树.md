# 7.5 AVL 树

AVL 树是**自平衡的 BST**，任意节点左右子树高度差（平衡因子）不超过 1，保证 O(log n)。

## 平衡因子

```java
// 平衡因子 = 左子树高度 - 右子树高度，只能是 -1、0、1
int balanceFactor(TreeNode node) {
    if (node == null) return 0;
    return height(node.left) - height(node.right);
}

int height(TreeNode node) {
    if (node == null) return 0;
    return node.height;  // AVL 节点需要额外存高度
}
```

## 四种旋转

插入后平衡因子超出范围，需要旋转来修复：

### LL（左偏左侧）→ 右旋
```
    z              y
   /      右旋     / \
  y      ───→    x   z
 /
x
```
```java
TreeNode rightRotate(TreeNode z) {
    TreeNode y = z.left, t = y.right;
    y.right = z;  z.left = t;
    updateHeight(z);  updateHeight(y);
    return y;
}
```

### RR（右偏右侧）→ 左旋
```java
TreeNode leftRotate(TreeNode z) {
    TreeNode y = z.right, t = y.left;
    y.left = z;  z.right = t;
    updateHeight(z);  updateHeight(y);
    return y;
}
```

### LR（左偏右侧）→ 先左旋再右旋
```java
TreeNode leftRightRotate(TreeNode z) {
    z.left = leftRotate(z.left);
    return rightRotate(z);
}
```

### RL（右偏左侧）→ 先右旋再左旋
```java
TreeNode rightLeftRotate(TreeNode z) {
    z.right = rightRotate(z.right);
    return leftRotate(z);
}
```

## 旋转选择

```java
TreeNode rotate(TreeNode node) {
    int bf = balanceFactor(node);
    if (bf > 1) {  // 左偏
        if (balanceFactor(node.left) >= 0) return rightRotate(node);      // LL
        else return leftRightRotate(node);                                  // LR
    }
    if (bf < -1) {  // 右偏
        if (balanceFactor(node.right) <= 0) return leftRotate(node);       // RR
        else return rightLeftRotate(node);                                  // RL
    }
    return node;  // 已平衡
}
```

## 插入和删除

```java
// 插入：和 BST 一样，只是最后多一步旋转
TreeNode insert(TreeNode node, int val) {
    if (node == null) return new AVLNode(val);
    if (val < node.val) node.left  = insert(node.left, val);
    else if (val > node.val) node.right = insert(node.right, val);
    else return node;
    updateHeight(node);
    return rotate(node);  // 检查并旋转
}

// 删除：和 BST 一样，最后多一步旋转
TreeNode remove(TreeNode node, int val) {
    if (node == null) return null;
    if (val < node.val) node.left  = remove(node.left, val);
    else if (val > node.val) node.right = remove(node.right, val);
    else {
        if (node.left == null)  return node.right;
        if (node.right == null) return node.left;
        TreeNode min = node.right;
        while (min.left != null) min = min.left;
        node.val = min.val;
        node.right = remove(node.right, min.val);
    }
    updateHeight(node);
    return rotate(node);  // 检查并旋转
}
```

## AVL vs 普通 BST

| 操作 | 普通 BST 最坏 | AVL 树 |
|------|-------------|--------|
| 查找 | O(n) | O(log n) |
| 插入 | O(n) | O(log n) |
| 删除 | O(n) | O(log n) |

**口诀**：平衡因子超1就旋转，LL右旋，RR左旋，LR先左后右，RL先右后左。插入删除后记得更新高度！
