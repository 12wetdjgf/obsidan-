# 5.2 队列（Queue）

队列像排队买奶茶，先来先服务，**先进先出（FIFO）**。

## 常用操作

| 操作 | 说明 | 时间复杂度 |
|------|------|----------|
| offer() | 入队（队尾） | O(1) |
| poll() | 出队（队头） | O(1) |
| peek() | 查看队头 | O(1) |
| isEmpty() | 是否为空 | O(1) |

## 两种实现

### 用链表实现（推荐）

```java
class LinkedQueue {
    private ListNode front = null;  // 队头
    private ListNode rear = null;   // 队尾
    private int size = 0;

    public void offer(int val) {
        ListNode node = new ListNode(val);
        if (rear == null) front = node;
        else rear.next = node;
        rear = node;
        size++;
    }

    public int poll() {
        int val = front.val;
        front = front.next;
        if (front == null) rear = null;
        size--;
        return val;
    }

    public int peek() { return front.val; }
    public boolean isEmpty() { return size == 0; }
}
```

### 用循环数组实现

普通数组出队后前面的空间浪费了，循环数组像环形跑道解决这个问题：

```java
class ArrayQueue {
    private int[] nums;
    private int front = 0;  // 队头指针
    private int size = 0;

    public ArrayQueue(int capacity) { nums = new int[capacity]; }

    public void offer(int val) {
        int rear = (front + size) % nums.length;
        nums[rear] = val;
        size++;
    }

    public int poll() {
        int val = nums[front];
        front = (front + 1) % nums.length;  // 循环移动
        size--;
        return val;
    }

    public int peek() { return nums[front]; }
    public boolean isEmpty() { return size == 0; }
}
```

## 典型应用

- **BFS（广度优先搜索）**：层序遍历树和图
- **打印机队列**：先提交先打印
- **消息队列**：异步任务处理
