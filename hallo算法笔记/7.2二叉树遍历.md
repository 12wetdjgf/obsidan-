# 7.2 二叉树遍历

遍历就是按某种顺序访问树的每个节点，共四种方式。

## 深度优先遍历（DFS）

### 前序遍历：根 → 左 → 右

```java
void preOrder(TreeNode root) {
    if (root == null) return;
    System.out.print(root.val + " ");  // 先访问根
    preOrder(root.left);
    preOrder(root.right);
}
// 树：1-2-4-5-3  结果：1 2 4 5 3
```

### 中序遍历：左 → 根 → 右

```java
void inOrder(TreeNode root) {
    if (root == null) return;
    inOrder(root.left);
    System.out.print(root.val + " ");  // 中间访问根
    inOrder(root.right);
}
// 结果：4 2 5 1 3
// ⭐ 对BST做中序遍历 → 得到升序序列！
```

### 后序遍历：左 → 右 → 根

```java
void postOrder(TreeNode root) {
    if (root == null) return;
    postOrder(root.left);
    postOrder(root.right);
    System.out.print(root.val + " ");  // 最后访问根
}
// 结果：4 5 2 3 1
// 适合：计算树的大小、删除树
```

## 广度优先遍历（BFS / 层序遍历）

一层一层从左到右，用**队列**实现：

```java
void levelOrder(TreeNode root) {
    if (root == null) return;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);

    while (!q.isEmpty()) {
        TreeNode node = q.poll();
        System.out.print(node.val + " ");
        if (node.left  != null) q.offer(node.left);
        if (node.right != null) q.offer(node.right);
    }
}
// 结果：1 2 3 4 5
```

## 迭代实现前序遍历（用栈模拟递归）

```java
void preOrderIter(TreeNode root) {
    Stack<TreeNode> stack = new Stack<>();
    if (root != null) stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        System.out.print(node.val + " ");
        // 先压右再压左，这样左边先弹出
        if (node.right != null) stack.push(node.right);
        if (node.left  != null) stack.push(node.left);
    }
}
```

## 四种遍历对比

| 遍历方式 | 顺序 | 结果（示例树） | 典型用途 |
|---------|------|-------------|--------|
| 前序 | 根左右 | 1 2 4 5 3 | 复制树、序列化 |
| 中序 | 左根右 | 4 2 5 1 3 | BST排序输出 |
| 后序 | 左右根 | 4 5 2 3 1 | 计算大小、删除 |
| 层序 | 逐层 | 1 2 3 4 5 | 找最短路、最小深度 |
