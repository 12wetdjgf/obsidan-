# 7.1 二叉树（Binary Tree）

树是由节点和边组成的层级结构，**二叉树**每个节点最多有两个子节点（左、右）。

## 基本术语

```
        1         ← 根节点（无父节点）
       / \
      2   3       ← 内部节点
     / \
    4   5         ← 叶节点（无子节点）
```

| 术语 | 说明 |
|------|------|
| 根节点 | 最顶层，无父节点 |
| 叶节点 | 无子节点 |
| 高度 | 根到最远叶节点的层数 |
| 深度 | 根到某节点的层数 |

## 节点定义

```java
class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int val) { this.val = val; }
}
```

## 创建一棵树

```java
TreeNode root = new TreeNode(1);
root.left  = new TreeNode(2);
root.right = new TreeNode(3);
root.left.left  = new TreeNode(4);
root.left.right = new TreeNode(5);
//        1
//       / \
//      2   3
//     / \
//    4   5
```

## 常见特殊二叉树

| 类型 | 特征 |
|------|------|
| 完美二叉树 | 所有叶节点在同一层，节点数 = 2^h - 1 |
| 完全二叉树 | 最后一层叶节点靠左对齐 |
| 完满二叉树 | 每个节点要么0个要么2个子节点 |
| 平衡二叉树 | 任意节点左右子树高度差 ≤ 1 |

## 基本操作

```java
// 插入节点（在 p 和 p.left 之间插入 n）
void insert(TreeNode p, TreeNode n) {
    n.left = p.left;
    p.left = n;
}

// 删除节点（删除 p.left，即节点 n）
void remove(TreeNode p) {
    p.left = p.left.left;  // 假设 n 只有左子节点
}
```

**口诀**：二叉树每节点最多俩娃，完美满，完全靠左，平衡高度差不超1。
