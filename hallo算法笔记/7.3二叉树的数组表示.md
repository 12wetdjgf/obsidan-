# 7.3 二叉树的数组表示

除了用节点+指针表示树，还可以用**数组**来表示——特别适合完全二叉树！

## 规则

把树按层序存入数组，下标从 0 开始：

```
        1           层序：1 2 3 4 5 6 7
       / \
      2   3         数组：[1, 2, 3, 4, 5, 6, 7]
     / \ / \        下标： 0  1  2  3  4  5  6
    4  5 6  7
```

对于下标 `i` 的节点：

```java
int parent(int i) { return (i - 1) / 2; }  // 父节点
int left(int i)   { return 2 * i + 1; }    // 左子节点
int right(int i)  { return 2 * i + 2; }    // 右子节点
```

## 含空节点的情况

非完全二叉树用 `null` 或特殊值表示空位：

```
        1
       / \
      2   3
       \
        5

数组：[1, 2, 3, null, 5, null, null]
下标： 0  1  2    3   4    5     6
```

```java
// 从数组构建二叉树
TreeNode buildTree(Integer[] arr) {
    if (arr == null || arr.length == 0 || arr[0] == null) return null;
    TreeNode root = new TreeNode(arr[0]);
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    int i = 1;
    while (!queue.isEmpty() && i < arr.length) {
        TreeNode node = queue.poll();
        if (i < arr.length && arr[i] != null) {
            node.left = new TreeNode(arr[i]);
            queue.offer(node.left);
        }
        i++;
        if (i < arr.length && arr[i] != null) {
            node.right = new TreeNode(arr[i]);
            queue.offer(node.right);
        }
        i++;
    }
    return root;
}
```

## 数组 vs 链表表示对比

| | 数组表示 | 链表（指针）表示 |
|--|---------|---------------|
| 空间 | 可能有空洞，浪费 | 只存实际节点 |
| 随机访问 | O(1) 定位父子节点 | 需要遍历 |
| 适合场景 | 完全二叉树（堆！） | 一般二叉树 |
| 插入删除 | 复杂 | 灵活 |

## 应用：堆（Heap）

堆就是用数组表示的完全二叉树！因为完全二叉树没有空洞，数组不浪费空间。

```java
// 堆的核心：通过下标公式快速找父子
int[] heap = {9, 8, 7, 6, 5, 4, 3};
//             0  1  2  3  4  5  6

// heap[1] 的父节点是 heap[(1-1)/2] = heap[0] = 9 ✓
// heap[0] 的左子节点是 heap[2*0+1] = heap[1] = 8 ✓
```

**口诀**：数组存完全二叉树，左孩 2i+1，右孩 2i+2，父节点 (i-1)/2。堆就是这么干的！
