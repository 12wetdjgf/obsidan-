# 9.1 图（Graph）

图比树更自由，任意两个节点之间都可以有边。

## 基本概念

| 术语 | 说明 |
|------|------|
| 顶点 Vertex | 图中的节点 |
| 边 Edge | 两顶点之间的连接 |
| 有向图 | 边有方向，A→B 和 B→A 不同 |
| 无向图 | 边无方向，A-B 和 B-A 相同 |
| 权重图 | 边有数值（距离、花费等） |
| 度 | 一个顶点连接的边数 |
| 入度/出度 | 有向图中指向/离开该节点的边数 |
| 连通图 | 任意两顶点之间都有路径 |

## 两种存储方式

### 邻接矩阵

```java
// matrix[i][j] = 1 表示 i 和 j 之间有边
int[][] matrix = {
    {0, 1, 1, 0},
    {1, 0, 1, 0},
    {1, 1, 0, 1},
    {0, 0, 1, 0}
};
// 判断边存在：O(1)，空间 O(V^2)，适合稠密图
```

### 邻接表（常用）

```java
Map<Integer, List<Integer>> graph = new HashMap<>();
graph.put(0, Arrays.asList(1, 2));
graph.put(1, Arrays.asList(0, 2));
graph.put(2, Arrays.asList(0, 1, 3));
graph.put(3, Arrays.asList(2));
// 空间 O(V+E)，适合稀疏图（现实中更常见）
```

## 邻接矩阵 vs 邻接表

| | 邻接矩阵 | 邻接表 |
|--|---------|--------|
| 空间 | O(V^2) | O(V+E) |
| 判断边存在 | O(1) | O(degree) |
| 获取所有邻居 | O(V) | O(degree) |
| 适合 | 稠密图 | 稀疏图 |

**口诀**：图=顶点+边，邻接矩阵快判断，邻接表省空间，稀疏图首选邻接表。
