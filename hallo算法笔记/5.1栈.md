# 5.1 栈（Stack）

栈就像一摞盘子，只能从顶部放和拿，**后进先出（LIFO）**。

## 常用操作

| 操作 | 说明 | 时间复杂度 |
|------|------|----------|
| push() | 压入栈顶 | O(1) |
| pop() | 弹出栈顶 | O(1) |
| peek() | 查看栈顶（不删） | O(1) |
| isEmpty() | 是否为空 | O(1) |

## 两种实现

### 用数组实现

```java
class ArrayStack {
    private ArrayList<Integer> stack = new ArrayList<>();

    public void push(int val) { stack.add(val); }

    public int pop() {
        if (isEmpty()) throw new RuntimeException("栈为空");
        return stack.remove(stack.size() - 1);
    }

    public int peek() { return stack.get(stack.size() - 1); }
    public boolean isEmpty() { return stack.isEmpty(); }
    public int size() { return stack.size(); }
}
```

### 用链表实现

```java
class LinkedStack {
    private ListNode top = null;
    private int size = 0;

    public void push(int val) {
        ListNode node = new ListNode(val);
        node.next = top;
        top = node;
        size++;
    }

    public int pop() {
        int val = top.val;
        top = top.next;
        size--;
        return val;
    }

    public int peek() { return top.val; }
    public boolean isEmpty() { return top == null; }
}
```

## 两种实现对比

| | 数组实现 | 链表实现 |
|--|---------|--------|
| 空间效率 | 可能有空闲空间 | 每个节点有指针开销 |
| 扩容 | 需要扩容操作 | 灵活，无需扩容 |
| 缓存友好 | ✅ 连续内存 | ❌ 跳跃访问 |

## 典型应用

- **括号匹配**：遇到左括号入栈，遇到右括号出栈对比
- **浏览器返回**：访问页面入栈，点返回出栈
- **撤销功能**：操作入栈，撤销时出栈
- **函数调用栈**：程序运行时自动维护
