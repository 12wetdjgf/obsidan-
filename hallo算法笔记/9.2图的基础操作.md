# 9.2 图的基础操作

## 基于邻接表的实现

```java
class GraphAdjList {
    Map<Integer, List<Integer>> adjList = new HashMap<>();

    // 添加顶点
    void addVertex(int v) {
        adjList.putIfAbsent(v, new ArrayList<>());
    }

    // 删除顶点
    void removeVertex(int v) {
        adjList.remove(v);
        for (List<Integer> neighbors : adjList.values())
            neighbors.remove(Integer.valueOf(v));
    }

    // 添加边（无向图）
    void addEdge(int u, int v) {
        adjList.computeIfAbsent(u, k -> new ArrayList<>()).add(v);
        adjList.computeIfAbsent(v, k -> new ArrayList<>()).add(u);
    }

    // 删除边
    void removeEdge(int u, int v) {
        adjList.getOrDefault(u, new ArrayList<>()).remove(Integer.valueOf(v));
        adjList.getOrDefault(v, new ArrayList<>()).remove(Integer.valueOf(u));
    }

    // 获取邻居
    List<Integer> neighbors(int v) {
        return adjList.getOrDefault(v, new ArrayList<>());
    }

    // 打印图
    void print() {
        for (Map.Entry<Integer, List<Integer>> e : adjList.entrySet())
            System.out.println(e.getKey() + " -> " + e.getValue());
    }
}
```

## 基于邻接矩阵的实现

```java
class GraphAdjMatrix {
    int[] vertices;
    int[][] adjMat;

    GraphAdjMatrix(int[] vertices, int[][] edges) {
        this.vertices = vertices;
        int n = vertices.length;
        adjMat = new int[n][n];
        for (int[] e : edges) addEdge(e[0], e[1]);
    }

    void addEdge(int i, int j) {
        adjMat[i][j] = 1;
        adjMat[j][i] = 1;  // 无向图
    }

    void removeEdge(int i, int j) {
        adjMat[i][j] = 0;
        adjMat[j][i] = 0;
    }

    boolean hasEdge(int i, int j) { return adjMat[i][j] == 1; }
}
```

## 操作复杂度对比

| 操作 | 邻接矩阵 | 邻接表 |
|------|---------|--------|
| 添加顶点 | O(V^2) 需扩矩阵 | O(1) |
| 删除顶点 | O(V^2) | O(V+E) |
| 添加边 | O(1) | O(1) |
| 删除边 | O(1) | O(degree) |
| 判断边存在 | O(1) | O(degree) |

**口诀**：邻接表添删顶点灵活，邻接矩阵判断边超快，按需选择。
