# 8.1 堆（Heap）

堆是一棵**完全二叉树**，分两种：
- **大顶堆**：每个节点 ≥ 子节点，根是最大值
- **小顶堆**：每个节点 ≤ 子节点，根是最小值

```
大顶堆：          小顶堆：
      9                 1
     / \               / \
    8   7             2   3
   / \ / \           / \ / \
  6  5 4  3         4  5 6  7
```

## 用数组存堆

堆是完全二叉树，完美契合数组存储：

```java
// 下标 i 的节点关系：
int left(int i)   { return 2 * i + 1; }
int right(int i)  { return 2 * i + 2; }
int parent(int i) { return (i - 1) / 2; }
```

## 堆的基本操作

### 入堆（push）：加末尾 → 上浮

```java
void push(int val) {
    heap.add(val);           // 加到末尾
    siftUp(heap.size() - 1); // 上浮
}

void siftUp(int i) {
    while (i > 0) {
        int p = parent(i);
        if (heap.get(p) < heap.get(i)) {  // 父节点比我小（大顶堆）
            swap(i, p);
            i = p;
        } else break;
    }
}
```

### 出堆（pop）：换末尾 → 删末尾 → 下沉

```java
int pop() {
    int top = heap.get(0);                  // 记录堆顶
    swap(0, heap.size() - 1);               // 堆顶和末尾交换
    heap.remove(heap.size() - 1);           // 删末尾
    siftDown(0);                            // 堆顶下沉
    return top;
}

void siftDown(int i) {
    int size = heap.size();
    while (true) {
        int largest = i, l = left(i), r = right(i);
        if (l < size && heap.get(l) > heap.get(largest)) largest = l;
        if (r < size && heap.get(r) > heap.get(largest)) largest = r;
        if (largest == i) break;
        swap(i, largest);
        i = largest;
    }
}
```

## Java 中用优先队列

```java
// 小顶堆（默认）
PriorityQueue<Integer> minHeap = new PriorityQueue<>();

// 大顶堆
PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a);

minHeap.offer(3);   // 入堆
minHeap.peek();     // 查看堆顶（不删）
minHeap.poll();     // 出堆（删除）
```

## 堆的时间复杂度

| 操作 | 时间复杂度 |
|------|----------|
| push 入堆 | O(log n) |
| pop 出堆 | O(log n) |
| peek 查顶 | O(1) |
| 建堆 | O(n) |

**口诀**：堆顶最值，push 上浮，pop 换末尾再下沉，Java 用 PriorityQueue，大顶堆传 (a,b)->b-a。
