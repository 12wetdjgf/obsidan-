# 9.3 图的遍历

图有两种遍历方式：BFS 广度优先 和 DFS 深度优先。

## BFS（广度优先搜索）- 一圈一圈扩散

就像往水里扔石头，一圈一圈往外扩。用**队列**实现。

```java
void bfs(Map<Integer, List<Integer>> graph, int start) {
    Set<Integer> visited = new HashSet<>();
    Queue<Integer> queue = new LinkedList<>();

    queue.offer(start);
    visited.add(start);

    while (!queue.isEmpty()) {
        int node = queue.poll();
        System.out.print(node + " ");  // 访问

        for (int neighbor : graph.getOrDefault(node, new ArrayList<>())) {
            if (!visited.contains(neighbor)) {
                visited.add(neighbor);
                queue.offer(neighbor);
            }
        }
    }
}
// 时间 O(V+E)，空间 O(V)
// 用途：最短路径（无权图）、层序遍历、连通分量
```

## DFS（深度优先搜索）- 一条道走到黑

就像走迷宫，一直往前走，撞墙了再回头换路。用**递归**（或栈）实现。

```java
void dfs(Map<Integer, List<Integer>> graph, int node, Set<Integer> visited) {
    visited.add(node);
    System.out.print(node + " ");  // 访问

    for (int neighbor : graph.getOrDefault(node, new ArrayList<>())) {
        if (!visited.contains(neighbor)) {
            dfs(graph, neighbor, visited);  // 递归深入
        }
    }
}

void dfsStart(Map<Integer, List<Integer>> graph, int start) {
    dfs(graph, start, new HashSet<>());
}
// 时间 O(V+E)，空间 O(V)
// 用途：拓扑排序、找环、路径搜索、连通分量
```

## 拓扑排序（Kahn 算法）

用于有向无环图（DAG），给出「先做什么后做什么」的顺序。
比如：课程依赖关系，先修完先修课才能修后续课。

```java
List<Integer> topoSort(Map<Integer, List<Integer>> graph, int n) {
    int[] inDegree = new int[n];
    for (int u : graph.keySet())
        for (int v : graph.get(u)) inDegree[v]++;

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < n; i++)
        if (inDegree[i] == 0) queue.offer(i);  // 入度为0的先入队

    List<Integer> order = new ArrayList<>();
    while (!queue.isEmpty()) {
        int node = queue.poll();
        order.add(node);
        for (int nb : graph.getOrDefault(node, new ArrayList<>())) {
            if (--inDegree[nb] == 0) queue.offer(nb);  // 入度减为0就入队
        }
    }
    // order.size() != n 说明有环！
    return order.size() == n ? order : new ArrayList<>();
}
```

## BFS vs DFS 对比

| | BFS | DFS |
|--|-----|-----|
| 数据结构 | 队列 | 递归/栈 |
| 遍历方式 | 一层一层扩散 | 一条路走到底 |
| 最短路径 | 适合（无权图） | 不适合 |
| 路径搜索 | 一般 | 适合 |
| 拓扑排序 | Kahn 算法 | DFS版本 |
| 内存 | 宽图耗内存大 | 深图递归栈深 |
