# 5.3 双向队列（Deque）

双向队列是队列的加强版，**两端都能插入和删除**，兼具栈和队列的功能！

## 常用操作

| 操作 | 说明 | 时间复杂度 |
|------|------|----------|
| addFirst() | 队头入队 | O(1) |
| addLast() | 队尾入队 | O(1) |
| removeFirst() | 队头出队 | O(1) |
| removeLast() | 队尾出队 | O(1) |
| peekFirst() | 查看队头 | O(1) |
| peekLast() | 查看队尾 | O(1) |

## 用双向链表实现

```java
class ListNode {
    int val;
    ListNode prev, next;
    ListNode(int v) { val = v; }
}

class LinkedDeque {
    private ListNode front, rear;
    private int size = 0;

    // 队头插入
    public void addFirst(int val) {
        ListNode node = new ListNode(val);
        if (isEmpty()) { front = rear = node; }
        else {
            node.next = front;
            front.prev = node;
            front = node;
        }
        size++;
    }

    // 队尾插入
    public void addLast(int val) {
        ListNode node = new ListNode(val);
        if (isEmpty()) { front = rear = node; }
        else {
            rear.next = node;
            node.prev = rear;
            rear = node;
        }
        size++;
    }

    // 队头删除
    public int removeFirst() {
        int val = front.val;
        front = front.next;
        if (front != null) front.prev = null;
        else rear = null;
        size--;
        return val;
    }

    // 队尾删除
    public int removeLast() {
        int val = rear.val;
        rear = rear.prev;
        if (rear != null) rear.next = null;
        else front = null;
        size--;
        return val;
    }

    public int peekFirst() { return front.val; }
    public int peekLast()  { return rear.val; }
    public boolean isEmpty() { return size == 0; }
}
```

## Java 中直接用

```java
Deque<Integer> deque = new ArrayDeque<>();

deque.addFirst(1);   // 队头插入
deque.addLast(2);    // 队尾插入
deque.peekFirst();   // 查看队头
deque.peekLast();    // 查看队尾
deque.removeFirst(); // 队头出队
deque.removeLast();  // 队尾出队
```

## 典型应用

**滑动窗口最大值**（超高频面试题！）

给数组和窗口大小 k，求每个窗口内的最大值。

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    Deque<Integer> dq = new ArrayDeque<>(); // 存下标，单调递减
    int[] res = new int[nums.length - k + 1];

    for (int i = 0; i < nums.length; i++) {
        // 队尾比当前小的全踢掉
        while (!dq.isEmpty() && nums[dq.peekLast()] <= nums[i])
            dq.removeLast();
        dq.addLast(i);

        // 队头已不在窗口内就踢掉
        if (dq.peekFirst() < i - k + 1) dq.removeFirst();

        // 窗口满了就记录最大值
        if (i >= k - 1) res[i - k + 1] = nums[dq.peekFirst()];
    }
    return res;
}
// 时间复杂度：O(n)，每个元素最多进出一次
```

**口诀**：双端队列两头都能操作，滑动窗口最爱它，维护单调性是关键。
