项目清单（优先可落地 + 有新意）

- Habit City Builder（把习惯变成城市）
    
    - 核心：每打勾一项任务，地图上自动长出建筑（学习=学院、运动=体育场）。
    
    - MVP：栅格地图 + 建筑贴图 + 任务→建筑规则 + 本地存储。
    
    - 拉满：随机事件（天气/节日加成）、城市导出海报、分享城市种子。
    

- LEGB Theater（CS61A 环境模型“剧场”）
    
    - 核心：输入一段 Python，自动生成环境图动画，函数/变量像角色演戏。
    
    - MVP：AST 解析 + 函数/变量帧的卡片动画（逐句高亮）+ 断点步进。
    
    - 拉满：典型错误的“戏剧化复现”、截图/动图导出、可嵌入博客教学。
    

- Meme DJ Tutor（鬼畜混剪 + 英语口语）
    
    - 核心：用 WebAudio 把中文/英文短句切片做节拍循环，同时做跟读训练。
    
    - MVP：音频加载与切片网格、节奏播放、录音比对（时长/能量）、得分。
    
    - 拉满：自动切片建议、节拍/和弦预设、作品一键导出。
    

- Code→Beat Sequencer（代码变音乐）
    
    - 核心：把代码的缩进、标识符长度、关键词序列映射成鼓点与旋律。
    
    - MVP：代码编辑框 + Tokenizer + 音序器播放 + 主题切换（Python/JS）。
    
    - 拉满：多轨导出、与 Git diff 结合“提交即音乐”、现场协作 Jam。
    

- Knowledge Garden（学习花园）
    
    - 核心：把你 Notion/笔记（可导出 JSON/Markdown）做成知识花园与时间线。
    
    - MVP：节点（文章/卡片）+ 边（引用/Tag）+ 时间轴播放，离线本地文件导入。
    
    - 拉满：检索高亮、复盘报告生成、CS61A 学习里程碑里嵌图谱片段。
    

- Vibe Coach（极小步任务生成器）
    
    - 核心：把“大目标”拆成“2分钟可做”的微任务，完成即即时反馈与动画。
    
    - MVP：预置拆解模板库（学习/运动/写作），自动拼接微任务 + 进度条。
    
    - 拉满：自学习模板（依据你的完成历史）、动机曲线提醒、节律音乐。
    

- Proof-of-Learning（学习凭证账本）
    
    - 核心：每次完成学习/项目生成一条“可分享的学习凭证卡”，形成时间账本。
    
    - MVP：卡片样式（课程/时长/证据截图）+ 本地签名（指纹）+ 导出/导入。
    
    - 拉满：简单“链上”模拟（本地 Merkle 证明）、简历嵌入组件。
    

- Shadow Speaking Studio（影子跟读工作室）
    
    - 核心：导入台词/字幕，自动分句计时；你跟读录音，对齐可视化反馈。
    
    - MVP：字幕时间轴 + 录音波形对齐（粗略）+ WPM/停顿反馈 + 连续得分。
    
    - 拉满：口音/连读“提示卡”、自定义台本库、学习打卡海报导出。
    

- Posture & Focus Coach（坐姿/专注教练）
    
    - 核心：长时间学习时给轻量提醒、简单坐姿检测（不存图像）+ 呼吸引导。
    
    - MVP：定时器 + 呼吸动画 + Webcam 姿态粗判（肩颈角度）+ 本地跑。
    
    - 拉满：专注音乐自适应、任务完成度联动“恢复值”、历史趋势。
    

- Project Arcade（微项目街机）
    
    - 核心：你预置一批精挑细选的小项目卡，现场可“抽卡挑战”，完成打勾出战报。
    
    - MVP：卡组（目标/预计时长/资源）+ 一键开始/结束 + 打包战报。
    
    - 拉满：挑战排行（本地）、“今日运势”加成、和 Habit City 互通奖励。
    

黑客松如何讲好故事

- 痛点：学习/习惯/项目输出太碎，缺乏“可见的进展与乐趣”。

- 方案：把“进度”转译成“可玩的东西”（城市、音乐、剧场、花园）。

- 亮点：离线单页、一键演示、可持续使用；不只是 Todo，而是“体验层的再设计”。

- 演示：固定 3 分钟脚本（问题→一分钟造物→互动→导出成果→Roadmap）。

快速落地建议

- 需要“稳交付”的创意：先选 Habit City Builder 或 LEGB Theater（易做又有记忆点）。

- 需要“爆点”的创意：Code→Beat 或 Meme DJ Tutor（舞台效果强）。

- 你若确定方向，我可立刻在当前目录生成对应的单文件离线 MVP 骨架（HTML/CSS/JS），含：
    
    - 基本交互与本地存储
    
    - 可演示的默认数据
    
    - 导出（JSON/图片/Markdown 其一）
    
    - 演示脚本提示
