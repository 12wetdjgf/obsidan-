## 📖 笔记内容速览

### 00_开篇：Agent就是一个死循环

**核心**：Agent = while True + stop_reason

```
用户 → LLM → 工具 → 结果 → LLM → ...
                ↑_____________↓
                    循环直到完成
```

**你会学到**：
- Agent的基本工作原理
- messages数组如何工作
- stop_reason的含义

---

### 01_最小化Agent循环

**核心**：一个工具 + 一个循环 = 完整的Agent

```python
while True:
    response = client.messages.create(...)
    if response.stop_reason != "tool_use":
        return
    # 执行工具
```

**你会学到**：
- 如何实现Agent循环
- 如何执行工具
- 如何处理工具结果

---

### 02_多工具系统与工具分发

**核心**：一个字典 + 一个查表 = 无限工具

```python
TOOL_HANDLERS = {
    "bash": run_bash,
    "read_file": run_read_file,
    "write_file": run_write_file,
}
handler = TOOL_HANDLERS[tool_name]
```

**你会学到**：
- 工具分发的设计模式
- 如何添加新工具
- 路径沙箱的实现

---

### 03_TodoWrite让Agent有计划

**核心**：TodoManager + 定期提醒 = 完成率翻倍

```python
if todo_manager.should_remind():
    inject_reminder()
```

**你会学到**：
- 为什么Agent会忘记任务
- 如何实现TodoManager
- 如何自动提醒

---

### 04_Subagents分解大任务

**核心**：子Agent + 上下文隔离 = 清洁的主Agent

def run_subagent(prompt):

**你会学到**：

- 如何防止无限递归
